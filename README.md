# Terra Vibe - Peneira Vibratória
No campo da arqueologia, muitos processos ainda dependem de ferramentas manuais que, embora eficientes, podem tornar o trabalho repetitivo, demorado e suscetível a variações humanas. A peneiração de sedimentos é um desses processos: essencial para identificar artefatos e microvestígios, mas frequentemente limitada por equipamentos improvisados ou difíceis de transportar.

Este projeto nasceu da necessidade de criar uma solução acessível, portátil e replicável para equipes de campo e laboratórios: uma peneira vibratória open source, com componentes de baixo custo. O objetivo é acelerar o processo, reduzir o esforço físico, melhorar a eficiência da triagem e aumentar a confiabilidade dos resultados.

A proposta permite que qualquer grupo possa construir, adaptar ou otimizar o equipamento de acordo com suas necessidades específicas. Ao tornar o projeto open source, pretendemos facilitar a colaboração entre arqueólogos, pesquisadores e desenvolvedores, promovendo melhorias contínuas e ampliando o acesso a ferramentas profissionais de peneiração.

## Features Principais

- Completamente gratuito e open source
- Estrutura fixa e resistente a base de canos PVC
- Não requer microcontroladores ou hardwares caros
- Possui ajuste de velocidade de peneiração para agilidade ou para fragmentos mais fragéis.
- Componentes acessíveis e fáceis de obter
- Não requer instalação de software/circuitos elaborados
- Funcionamento requer apenas uma tomada de 110v

## Como Funciona?

O processo é simples: o usuário distribui o material a ser peneirado (solo, sedimentos ou fragmentos arqueológicos) sobre a superfície da peneira, garantindo que esteja espalhado de forma uniforme. Logo abaixo da peneira, posiciona-se um recipiente de coleta, como um balde ou bandeja, que irá receber a terra, areia ou outros materiais indesejados que passarem pela malha durante o processo. Em seguida, com o TerraVibe ligado a tomada, basta ajustar o potenciômetro para definir a intensidade de vibração desejada. Essa vibração facilita a separação natural dos fragmentos: pedaços maiores (como artefatos, fragmentos cerâmicos ou pequenas estruturas) permanecem sobre a peneira para análise enquanto, os materiais indesejados caem para o recipiente de coleta através da malha.
